¿Que es un contenedor?
En HTML, un contenedor se refiere a un elemento que se utiliza para contener
y estructurar otros elementos dentro de una página web. Los contenedores son
elementos HTML que actúan como "cajas" que pueden contener otros elementos
como texto, imágenes, formularios, listas, tablas, entre otros.
